# ModX

Универсальный менеджер модов с автоматической поддержкой инъекторов (REFramework, BepInEx), встроенным веб-браузером и блокировщиком рекламы. 

## 🚀 Основные возможности

* **Умная библиотека игр**
  * Глубокий поиск игр (сканирование везде, по всем локальным дискам и папкам).
  * Ручное добавление `.exe` файлов.
  * Управление избранным для быстрого доступа.
  * Компактный и полноразмерный вид карточек игр.

* **Управление инъекторами (Injectors)**
  * Встроенная загрузка и обновление **REFramework** (ReEngine) напрямую с GitHub.
  * Встроенная загрузка и обновление **BepInEx** (Unity) с выбором версии.
  * Менеджер локальных версий — переключение и удаление в пару кликов.

* **Установка модов и скриптов**
  * Установка простым перетаскиванием (Drag & Drop) архивов (`.zip`, `.rar`, `.7z`) или файлов (`.pak`, `.lua`).
  * Автоматическое распределение по вкладкам "Моды|Плагины" и "Скрипты|Патчеры".
  * Чтение метаданных мода (`modinfo.ini`): вывод описания, версии, автора и обложки.
  * Безопасное включение/отключение модов с проверкой на конфликты файлов.

* **Встроенный браузер**
  * Многовкладочность и поддержка закладок.
  * **Встроенный AdBlocker**: жесткая блокировка рекламных сетей, трекеров и всплывающих окон.

* **Оптимизация**
  * Аппаратное ускорение графики (UI и WebView2).
  * Автоматическая очистка кэша браузера при выходе.

## ⚙️ Требования

* **ОС:** Windows 10 / 11
* **Платформа:** .NET 10.0 Desktop Runtime
* **Компоненты:** Microsoft Edge WebView2 Runtime

## 🛠 Установка

1. Скачайте последнюю версию со страницы релизов.
2. Распакуйте архив в любую удобную папку.
3. Запустите `ModX.exe`.

## 📖 Быстрый старт

1. **Добавьте игру:** На главном экране нажмите `ПОИСК ИГР` или добавьте исполняемый файл вручную.
2. **Настройте базу:** Кликните на игру. Во вкладке `Инъекторы` загрузите BepInEx или REFramework, если того требует игра.
3. **Установите моды:** Откройте вкладку `Моды|Плагины` и перетащите в окно скачанный архив. 
4. **Играйте:** Включите нужные моды тумблером слева и нажмите `ЗАПУСТИТЬ ИГРУ`.

## 💻 Технологический стек

* **UI/UX:** WPF, XAML, Windows 11 Dark Mode styling.
* **Браузерный движок:** `Microsoft.Web.WebView2`
* **Работа с архивами:** `SevenZipExtractor`
